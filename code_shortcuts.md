# VSCode Vim Shortcuts

> Leader key is `<space>`

---

## Insert Mode

| Keys | Description |
|------|-------------|
| `j j` | Exit insert mode (`<Esc>`) |

---

## Normal Mode — Ctrl Bindings

| Keys | Description |
|------|-------------|
| `C-h` | Show hover info |
| `C-i` | Peek implementation |
| `C-k` | Peek declaration |
| `C-l` | Clear search highlight |
| `C-t` | Navigate back |
